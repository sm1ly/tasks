Практическая задача на чтение файла CSV:

Темы:
1. Разбор аргументов командной строки
2. Чтение и парсинг CSV
3. Работа с исключениями

Дано CSV файл - каждая строка - несколько полей,
    разделённых каким-либо символом

Написать программу для чтения этого CSV-файла,
    проверки строк на предмет того, что один из столбцов
    содержит указанное значение, и вывода количества
    строк, которые под это условие попадает.

Дополнительные данные:
1. Первая строка файла содержит имена полей
2. Программа должна принимать имя файла из именованных аргументов командной строки
3. Условие так же принимается через аргументы командной строки
4. Вывод статистики на стандартный вывод

Подсказки:
1. Для парсинга аргументов командной строки удобнее использовать готовые модули.
Например, модуль argparse входит в стандартную библиотеку.
2. Аргументы командной строки:
 --input <FILE> - путь к CSV-файлу
 --fieldName <NAME> - имя столбца (поля)
 --fieldValue <VALUE> - значение столбца (поля)
3. Файл CSV сгенерировать самостоятельно
4. Предусмотреть обработку исключений, если файл не найден
